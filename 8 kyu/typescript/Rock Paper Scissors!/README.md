## Description

Rules of the "Rock, Paper, Scissors" game are:

- Rock beats Scissors
- Scissors beat Paper,
- Paper beats Rock.

Let's play! You have to return which player won! In case of a draw return Draw!.

Examples:

```plaintext
"scissors", "paper" --> "Player 1 won!"
"scissors", "rock" --> "Player 2 won!"
"paper", "paper" --> "Draw!"
```
